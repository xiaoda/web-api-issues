## Demo
本项目提供本地 Demo，包含 React 构建的前端网页及 Express 构建的后端接口。

#### 前端网页运行方法
```
cd react
npm install
npm start // 运行在 http://localhost:3721
```

#### 后端接口运行方法
```
cd express
npm install
npm run dev // 运行在 http://127.0.0.1:3824
```
